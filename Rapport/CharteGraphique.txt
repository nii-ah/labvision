BORDURES
----------------------------------------------------------------
// Deux niveaux de bordure selon la visibilité souhaitée :
// - Discrète pour les éléments de structure (cartes, séparateurs)
// - Standard pour les éléments interactifs (inputs, selects)
 
Épaisseur standard    1px       Cartes, inputs, dividers
Épaisseur focus       2px       Input en focus, élément sélectionné
 
Bordure discrète      #E5E7EB   Cartes, dividers, séparateurs
                                // Très légère, quasi invisible — pour la structure
Bordure standard      #7a969f   Inputs au repos, éléments interactifs
                                // Plus visible — signale que l'élément est cliquable
Bordure focus         #611677   Input en focus (primaire)
                                // Couleur primaire — indique l'élément actif
